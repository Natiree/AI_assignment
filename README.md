# AI Assignment

This repository contains my solutions for the Artificial Intelligence assignment. It has four tasks: MNIST digit recognition, constraint satisfaction problems, Prolog programming, and search algorithms.

---

## Task 1: MNIST Dataset

I downloaded the MNIST dataset and built a simple neural network using TensorFlow. The model learns to recognise handwritten digits from 0 to 9.

Files:
- task1a_mnistDownload.py
- task1b_digitRecognition.py

---

## Task 2: Constraint Satisfaction

### (a) Australia Map
I used constraint programming to colour regions of Australia. The rule is that no two neighbouring regions should have the same colour.

### (b) Nairobi Sub-counties
I applied the same idea to Nairobi sub-counties and assigned colours while making sure adjacent areas do not share the same colour.

Files:
- task2a_australiaColoring.py
- task2b_nairobiColoring.py

---

## Task 3: Prolog

I installed and used SWI-Prolog.

- Created simple parent and grandparent relationships
- Tested basic logic queries
- Built a family tree with different relationships like cousins and uncles

Files:
