# AI Assignment

This repo contains my solutions for the AI assignment. It includes four tasks: MNIST digit recognition, constraint satisfaction problems, Prolog programming, and search algorithms.

## Task 1: MNIST
I used TensorFlow to load the MNIST dataset and train a simple neural network to recognise handwritten digits (0–9).

Files: task1a_mnistDownload.py, task1b_digitDistinguish.py

## Task 2: Constraint Satisfaction
(a) Australia Map – I coloured regions of Australia using 3 colours, making sure no neighbouring regions share the same colour.

(b) Nairobi Sub-counties – I applied the same idea to Nairobi sub-counties and ensured adjacent areas do not use the same colour.

Files: task2a_australiaColouring.py, task2b_nairobiColouring.py

## Task 3: Prolog
I used SWI-Prolog to set up basic logic programming.
- Defined parent and grandparent relationships
- Tested simple queries
- Built a small family tree with relations like cousins and uncles

Files: task3b_family.pl, task3c_myFamily.pl

## Task 4: Search Algorithms
BFS (Breadth First Search) was used to find the shortest path in a graph, while DFS (Depth First Search) explores deeper paths first before backtracking.

Files: task4_bfs.py, task4_dfs.py

## How to run
Python files:
python filename.py

Prolog files:
Open SWI-Prolog and load:
[filename].
